Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

External CSS

Met een External css file sheet kan je de look van de website compleet veranderen door maar 1 link aan te passen in je html bestand

Internal CSS

Een Internal style sheet kan gebruikt worden als 1 html pagina een unieke stijl heeft. De Internal style sheet word herkent aan de <style> element in de <head> sectie van je html

Inline CSS

Een Inline style kan gebruikt worden voor een unieke stijl van een specifiek element Om inline stijlen te gebruiken, voegt u het style-attribuut toe aan het relevante element.
