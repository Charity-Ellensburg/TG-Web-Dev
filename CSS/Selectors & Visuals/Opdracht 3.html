Er zijn verschillende selectors, bijvoorbeeld;
  • type selectors
  • descendant selectors
  • class Selectors
Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

Type selectors

Een type selector wordt ook wel een tag name selector of een element selector genoemd, omdat het een HTML-tag / -element in uw document selecteert. Een voorbeeld hiervan is de <h1> selector

Descendant selectors

De descendant selector komt overeen met alle elementen die afstammelingen zijn van een gespecificeerd element. De eerste eenvoudige selector binnen deze selector vertegenwoordigt het voorouderelement - een structureel superieur element, zoals een ouderelement, of de ouder van een ouderelement, enzovoort.
