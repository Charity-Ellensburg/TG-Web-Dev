Opdracht 1

Welke position values zijn er? Leg voor elke position values uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

Er zijn 5 soorten position properties:

  • static
  • relative
  • fixed
  • absolute
  • sticky

Static

Relative
Een element met de positie relative word elders gepositioneerd ten opzichte van zijn normale positie. Door de properties top, bottom, left and right toe te passen word hij weg gehaald van zijn normale positie. De opening die eventueel word achter gelaten word niet opgevuld door andere inhoud.

Fixed
Een element met de positie fixed blijft altijd op dezelfde plaats zelf als de pagina word gescrolld. De properties top, right, bottom and left worden gebruikt om het element te positioneren. Een element met de position fixed laat geen gat achter waar hij normaal gesproken zou staan.

Absolute
Een element met positie absolute is gepositioneerd ten opzichte van de dichtstbijzijnde gepositioneerde ancestor. Als een absolute gepositioneerd element geen gepositioneerde voorouders heeft, gebruikt het de body van het document en beweegt het mee met het scrollen van de pagina.

Sticky
Een element met positie sticky wordt gepositioneerd op basis van de scrollpositie van de user. Een sticky element wisselt tussen relatif en static, afhankelijk van de scrollpositie. Het wordt relatief gepositioneerd totdat een bepaalde offset-positie wordt bereikt in de viewport dan blijft het op zijn plaats plakken zoals positie static.
