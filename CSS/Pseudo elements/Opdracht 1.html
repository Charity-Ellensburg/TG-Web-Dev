Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

pseudo elementen worden gebruikt om specifieke delen van een element te stijlen. Pseudo element ::firstletter wordt gebruikt om de eerste letter van een zin te stijlen.

Pseudo element ::firstline wordt gebruikt om de eerste regel van een alinea te stijlen.
