e. Wat komt er uit als je de + vervangt door:
  1. ==
    2. Hier worden de values met elkaar vergeleken maar het datatype genegeerd. Het is hier dus een Boolean met als antwoord true
  2. ===
    3. Hier vergelijkt hij de values maar ook het datatype. Nogmaals een boolean maar dan met antwoord false. De reden hiervoor is omdat someNumber een number is en anotherNumber is een string.
  3. !=
    4. Hier word er een vraag gesteld oftewel boolean met dat de twee values niet gelijk zijn aan elkaar. Dat is in dit geval waar dus het antwoord is dan ook true.
  4. !==
    5. Hier gebeurt eigenlijk het zelfde alleen word er naast het vergelijken van de values ook het datatype vergeleken en dat is in dit geval ook niet gelijk dus het antwoord is weer tre.
