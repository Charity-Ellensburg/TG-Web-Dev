Zoek op of experimenteer wat += doet. Werkt deze ook met Strings? Weet je hoe je deze anders kan schrijven?

de addition assignment voegt waarde toe aan een variable. Het werkt met number en strings. Je kan de syntax ook zo opschrijven --> firstName + lastName =

Doe hetzelfde voor -=, *=, /=, %= en **=

-= Werkt met numbers maar niet voor strings. De syntax kan ook zo worden opgeschreven --> example - newExample =

*= werkt met numbers maar niet voor strings. De syntax kan ook zo worden opgeschreven --> example * newExample =

/= werkt met numbers maar niet voor strings. De syntax kan ook zo worden opgeschreven --> example / newExample =

%= werkt met numbers maar niet voor strings. De syntax kan ook zo worden opgeschreven --> example % newExample =

**= werkt met numbers maar niet voor strings. De syntax kan ook zo worden opgeschreven --> example ** newExample =
