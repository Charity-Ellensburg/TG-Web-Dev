A. Wat doet substring als het eerste getal hoger is dan het tweede getal?

Hij wisselt dan index om. Het 2e getal gaat dan over start en het 1e getal over stop.

B. Wat doet slice als het eerste getal hoger is dan het tweede getal?

Je krijgt dan een empty string terug.

C. Wat doet substring bij negatieve waardes?

Je krijgt een empty string terug

D. Wat doet slice bij negatieve waardes?

Als beide waardes negatief zijn krijg je een empry string. Als alleen stop negatief is trekt hij bij wijze van spreken die waarde van de string af en daar stopt hij dan de index. bijv: slice: 15, -1 "buurman heeft kittens gehad". slice: 15, -7: "Buurman heeft kittens."
